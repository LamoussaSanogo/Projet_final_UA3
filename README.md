## 🎯 Objectif du projet

Ce projet consiste à développer un programme Java permettant de :

* Lire les données des étudiants à partir d’un fichier CSV
* Calculer la moyenne de chaque étudiant
* Classer les étudiants par ordre décroissant de moyenne
* Sauvegarder les résultats dans un autre fichier CSV

---

## 🧱 Structure du projet

* Main.java
* Etudiant.java
* LectureCSV.java
* GestionNotes.java
* SauvegardeCSV.java

---

## 📁 Fichiers utilisés

## ▶️ Exécution du programme

1. Ouvrir le projet dans IntelliJ IDEA
2. Vérifier que le fichier `notes.csv` est placé à la racine du projet
3. Exécuter la classe `Main.java`
4. Utiliser le menu pour tester les différentes fonctionnalités
